## ✨ What is Prompt Navigator?

Prompt Navigator is a powerful browser extension designed for AI enthusiasts, researchers, and professionals who work with multiple AI platforms daily. It transforms how you interact with ChatGPT, Claude, Gemini, Perplexity, and Poe by providing seamless prompt management, smart organization, and cross-platform synchronization.

### 🎯 Why Choose Prompt Navigator?

- **🌐 Universal Compatibility** - Works across all major AI platforms
- **⚡ Lightning Fast** - Instant access to your prompt library
- **🔐 Privacy First** - All data stays in your browser
- **🎨 Beautiful Interface** - Modern, intuitive design
- **📊 Smart Analytics** - Track and optimize your prompt usage

## 🚀 Installation

### 🟦 Chrome / Edge / Brave

```bash
# Method 1: Chrome Web Store (Recommended)
Visit: chrome://extensions/ → Search "Prompt Navigator" → Install

# Method 2: Developer Mode
1. git clone https://github.com/yourusername/prompt-navigator.git
2. Open chrome://extensions/
3. Enable "Developer mode" (top-right toggle)
4. Click "Load unpacked" → Select extension folder
5. Pin extension to toolbar for quick access
```

### 🟧 Firefox

```bash
# Method 1: Firefox Add-ons (Recommended)
Visit: about:addons → Search "Prompt Navigator" → Install

# Method 2: Temporary Installation
1. git clone https://github.com/yourusername/prompt-navigator.git
2. Open about:debugging#/runtime/this-firefox
3. Click "Load Temporary Add-on"
4. Select manifest.json from the extension folder
```

Safari support is in development. Follow our [roadmap](#roadmap) for updates.

## 📖 Quick Start Guide

### 1️⃣ First Launch

1. **Install** the extension following the steps above
2. **Visit** any supported AI platform
3. **Click** the Prompt Navigator icon in your browser toolbar
4. **Welcome Tour** - Take the interactive tutorial (recommended)

### 2️⃣ Basic Usage

```markdown
🎯 QUICK ACTIONS
┌─────────────────────────────────────┐
│ Ctrl/Cmd + Shift + P → Toggle Panel │
│ Ctrl/Cmd + S        → Save Prompt   │
│ Ctrl/Cmd + F        → Search        │
│ Escape              → Close Panel   │
└─────────────────────────────────────┘
```

### 3️⃣ Saving Your First Prompt

1. **Type** or select text in any AI chat
2. **Right-click** → "Save as Prompt"
3. **Add** title, description, and tags
4. **Choose** category or create new one
5. **Save** - Your prompt is now in your library!

## 🚨 Troubleshooting

### ⚠️ Common Issues

<details>
<summary><b>🔧 Extension Not Loading</b></summary>

**Symptoms:** Extension icon missing or non-functional

**Solutions:**
1. Check browser compatibility (Chrome 88+, Firefox 78+)
2. Verify all files are present in extension directory
3. Validate `manifest.json` syntax
4. Check browser console for error messages
5. Try disabling other extensions temporarily
6. Reload extension: `chrome://extensions/` → Click reload

**Advanced Debugging:**
```bash
# Check extension logs
chrome://extensions/ → Developer mode → Errors → View details
```

</details>

<details>
<summary><b>🎯 Navigation Panel Issues</b></summary>

**Symptoms:** Panel not appearing or malfunctioning

**Solutions:**
1. Ensure you're on a supported platform
2. Check if content scripts loaded (F12 → Console)
3. Refresh the page completely
4. Clear browser cache and cookies
5. Check for JavaScript errors

**Manual Fix:**
```javascript
// Run in console to force reload
location.reload(true);
```

</details>

<details>
<summary><b>💾 Storage Problems</b></summary>

**Symptoms:** Prompts not saving or disappearing

**Solutions:**
1. Check storage permissions in extension settings
2. Verify available storage space
3. Clear extension data: Settings → Extensions → Prompt Navigator → Site data
4. Export prompts before reinstalling

**Storage Check:**
```javascript
// Check storage usage (run in extension popup console)
chrome.storage.local.getBytesInUse(null, (bytes) => {
  console.log('Storage used:', bytes, 'bytes');
});
```

</details>
